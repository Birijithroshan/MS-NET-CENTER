# MS Net Centre

> Your One-Stop Solution for All Online Services & Document Needs

**MS Net Centre** is the website for a fully authorized e-Sevai centre located in **Nochimedu, Manjampatti, Manapparai, Trichy**. It showcases the wide range of government, digital, and document services offered to the local community.

## Features

- **Responsive Design** — works on desktops, tablets, and mobile devices
- **Animated Preloader** with a computer-themed SVG graphic
- **Scroll Animations** — counter animation, scroll-reveal cards, and parallax particles
- **Mobile Navigation** with a hamburger toggle menu
- **Contact Form** that composes a WhatsApp message for instant enquiry
- **Floating WhatsApp Button** for quick customer support
- **Back-to-Top Button** for easy page navigation
- **Interactive Card Tilt Effect** on service cards

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox/grid layouts, keyframe animations
- **JavaScript** (vanilla) — Intersection Observer API, DOM manipulation
- **Google Fonts** — Orbitron, Poppins, Rajdhani, Raleway
- **Font Awesome 6** — icon library

## Getting Started

No build tools or dependencies are required. Simply open the site in any modern browser:

```bash
# Clone the repository
git clone https://github.com/Birijithroshan/MS-NET-CENTER.git

# Open in browser
open MS-NET-CENTER/index.html
```

Or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000` in your browser.
