# MS Net Centre - Website

A responsive single-page website for **MS Net Centre**, an e-Sevai centre located in Nochimedu, Manjampatti, Manapparai, Trichy.

## About

MS Net Centre provides online services including government e-Sevai applications, printouts, xerox, and other digital services to the local community.

## Tech Stack

- **HTML5** — Page structure and content
- **CSS3** — Styling with custom properties, animations, and responsive design
- **JavaScript** — Interactive features (preloader, scroll effects, mobile navigation, counters)
- **Font Awesome** — Icons
- **Google Fonts** — Orbitron, Poppins, Rajdhani, Raleway

## Features

- Responsive design for all screen sizes
- Animated preloader
- Sticky navbar with scroll effects
- Mobile hamburger menu
- Smooth scrolling navigation
- WhatsApp floating contact button
- Service & pricing sections
- Gallery section
- Contact section

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in a web browser.

No build tools or dependencies required — it's a static website.
